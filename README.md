# MiTienda - Sistema de Gestión de Productos

## Descripción

MiTienda es una aplicación de escritorio desarrollada en JavaFX que permite gestionar productos mediante operaciones CRUD (Crear, Leer, Actualizar y Eliminar). El sistema cuenta con una interfaz gráfica moderna diseñada con CSS, autenticación de usuarios y una tabla para visualizar los productos registrados.

Este proyecto fue desarrollado con fines académicos para aplicar conceptos de Programación Orientada a Objetos, interfaces gráficas en JavaFX y manejo de eventos.

---

## Objetivo General

Desarrollar una aplicación de escritorio que permita administrar productos de manera eficiente mediante una interfaz gráfica amigable utilizando JavaFX y CSS.

---

## Objetivos Específicos

- Implementar un sistema de inicio de sesión para controlar el acceso a la aplicación.
- Aplicar operaciones CRUD para la gestión de productos.
- Diseñar una interfaz gráfica moderna y funcional utilizando JavaFX y CSS.

---

## Funcionalidades

### Inicio de Sesión
- Validación de usuario.
- Selección de rol.
- Acceso al sistema mediante credenciales.

### Gestión de Productos
- Registrar productos.
- Actualizar información de productos.
- Eliminar productos.
- Limpiar formulario.
- Visualizar productos en una tabla.

### Interfaz Gráfica
- Diseño moderno mediante CSS.
- Menú lateral de navegación.
- Formularios organizados.
- Tabla de datos estilizada.
